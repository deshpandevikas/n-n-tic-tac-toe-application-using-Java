1. Application Requirements:

If you are checking on a desktop or a laptop, you need android studio 2.1.3 (or any igher version) to be installed.
"Android platform with the minimum SDK 17 (android 4.2) and a maximum SDK  23 (android 6.0)"


2. How to run our program?

a. In case of android studio, Download and import the project to android studio and click on ‘Run’ which launches the emulator which in turn opens the application.
b. In case of having a cell phone with Android operating system, install the apk file and open.
The apk will be available at "ISTicTacToe/app/build/outputs/apk"
